Grupo/Cliente

Grupo/Cliente é a entidade que agrupa os CNPJS de diferentes empresas. Uma entidade genérica para relacionar as diferentes empresas de um grupo com o objetivo de simplificar a operação.


Esta funcionalidade é mais voltada para profissionais/empresas que irão gerir carteiras de múltiplas empresas no plano Premium. Caso você esteja gerenciando apenas uma empresa, pode usar o mesmo nome para o Grupo e para a empresa.

Habilite esta funcionalidade!

Para cadastrar mais de um Grupo/Cliente é necessário que sua assinatura seja do plano Premium ou Pro. Você não gostaria de desbloquear esta e outras funcionalidades que esses planos oferecem ? Faça seu upgrade agora!

Envio automático de eventos do eSocial

É possível configurar para incluir os eventos de SST dos funcionários desse cliente automaticamente em lotes de envio ao eSocial.

Caso escolha ativar essa funcionalidade, a cada 12 horas o sistema procurará os eventos disponíveis para envio de todos os funcionários de todas as empresas desse cliente e analisará se é possível incluir esses eventos em lotes de envio ao eSocial.

Essa análise prévia indicará se há inconsistências sistêmicas ou não. Caso não exista impedimento o evento será incluído em um lote de envio, que entrará em uma fila de envio executada a cada 40 minutos.

Caso o processo de envio automático encontre algum impedimento para inclusão do evento em um lote, o e-mail do responsável (definido no menu Customização -> Dados da Assinatura) receberá uma mensagem de aviso e poderá analisar e resolver as pendências indicadas no menu Gestão do eSocial -> Relatório de erros.
